## Procedura Consigliata per il Deposito

### 1. Deposito Digitale (Veloce e Internazionale) - RACCOMANDATO
Utilizza **Safe Creative** o **Copyright.eu**:
1. Crea un file `.zip` di questa intera cartella `COPYRIGHT_DEPOSIT_2024`.
2. Carica lo zip sul sito scelto.
3. Riceverai un certificato con "marcatura temporale" (Timestamp) che prova legalmente che in questa data tu possedevi questo codice.

### 2. Deposito SIAE (Formale per l'Italia)
Se desideri una protezione istituzionale italiana:
1. Copia il contenuto di questa cartella su una chiavetta USB o un DVD (non riscrivibile).
2. Compila il **Modello 349** della SIAE (Pubblico Registro Software).
3. Consegna il supporto e il modulo seguendo le istruzioni sul sito ufficiale SIAE.

### 3. Notarizzazione Blockchain (Tecnologica)
Utilizza servizi come **Proofy**:
1. Carica il file `.zip` per ottenere lo "hash" salvato su Blockchain.
2. È una prova matematica immutabile della creazione dell'opera.
